## ZTALock: Zero Trust Network Architecture (Cisco Packet Tracer)

This repository contains my Zero Trust Network project built in Cisco Packet Tracer.

For full documentation, screenshots, diagrams, and explanations, visit the project website:

Project Website (GitHub Pages):

About the Project

This project demonstrates a Zero Trust Network Architecture using:

VLAN-based segmentation
ACL microsegmentation
DMZ isolation
DHCP snooping
Port security
NAT overload (PAT) for Internet access
Centralized Syslog logging
SSH access restricted to the management VLAN

All testing results and explanations are available on the project website.
